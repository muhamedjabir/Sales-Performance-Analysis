# Sales Performance Analysis Dashboard #
Project Overview

This project analyzes retail sales data to understand business performance across products, categories, cities, and shipping modes. The analysis was performed using SQL, and insights were visualized through an interactive Power BI dashboard.

The goal of this project is to demonstrate practical data analysis, SQL querying, and dashboard development skills.

## Tools & Technologies ##

- SQL (MySQL) – Data analysis and querying

- Power BI – Interactive dashboard and data visualization

- Excel – Dataset source

## Dataset ##

The project uses the Superstore sales dataset, which contains transactional data including:

- Order Date

- Product Name

- Category and Sub-Category

- Sales and Profit

- Quantity

- City and Region

- Shipping Mode

This dataset helps analyze sales performance, product demand, and profitability trends.

## Dashboard Pages ##
1. Sales Overview

Displays key performance indicators such as:

- Total Sales

- Total Profit

- Quantity Sold

- Total Orders

- Average Order Value

This page provides a high-level view of overall business performance.

2. Product Analysis

Analyzes sales and profit across categories and sub-categories.
Drill-down functionality allows deeper exploration of product performance.

3. Regional & Shipping Analysis

Shows sales distribution across cities and shipping modes using treemap and charts.

4. Product Details

A searchable product-level table that allows users to quickly find and analyze individual product performance.

## Key Insights ##

- Technology category generates the highest sales and profit.

- Sales are concentrated in a few major cities.

- A small number of products contribute significantly to overall revenue.

- Standard shipping is the most commonly used delivery mode.

- Some products generate high sales but relatively lower profit margins.

  ## SQL Analysis ##

- SQL queries were used to perform analytical operations such as:

- Total sales and profit calculations

- Top-performing cities by sales

- Product sales ranking

- Running total of sales over time

- Category-level performance analysis

- These queries helped generate insights that were visualized in the Power BI dashboard.
